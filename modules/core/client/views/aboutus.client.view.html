Martin McLaughlin

My name is Martin McLaughlin. I've always had an interest in computers, and as soon as I got my own I was taking it apart and putting it back together. Programming was a perfect fit for my inquisitive mind and I started coding and making games when I was about 15. Nowadays most of my coding is college related, but I look forward to making stuff for myself a bit more when I have the time. I found Angular very challenging to learn, but once I wrapped my head around the file structure and the MV* model it became a lot easier. Overall, I enjoy using JavaScript despite its quirks, so being a Full-Stack JS developer could be fun!
